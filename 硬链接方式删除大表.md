大表直接drop删除可能会造成`hang住`的情况，可以采用硬链接方式进行删除。
**操作前提是数据库开启了独立表空间**
```
mysql> show global variables like '%innodb_file_per_table%';
+-----------------------+-------+
| Variable_name         | Value |
+-----------------------+-------+
| innodb_file_per_table | ON    |
+-----------------------+-------+
```
